# Software Fellowship

This repository contains the projects shown during the Software Fellowship program.

## Projects

### Expense Tracker

A simple expense tracking application built to practice web development concepts.

**Features**

- Add expenses
- View expense list
- delete expenses

---

### Flex Model

A demonstration project for understanding **CSS Flexbox**.

This project was presented during the fellowship presentation and is intended for experimentation with different Flexbox properties.

You can try changing properties such as:

```css
justify-content:
align-items:
flex-direction:
flex-wrap:
gap:
```

Example:

```css
.mydiv {
  display: flex;
  justify-content: space-evenly;
  align-items: center;
}
```

Experiment with different values to observe how the layout changes.

## Getting Started

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open either project folder.

3. Run the HTML files in your browser.

Example:

```bash
cd FlexModel
```

Then open `index.html` in your browser.
